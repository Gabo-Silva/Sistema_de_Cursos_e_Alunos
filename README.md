# Sistema_de_Cursos_e_Alunos

# Introdução

Nesse sistema você pode cadastrar, listar e remover alunos e cursos, os alunos podem ser matriculados em diversos cursos, e há também a possibilidade de caso o usuário queira de cancelar a matricula do aluno.

Eu utilizei python como liguagem e sua biblioteca de banco de dados, o sqlite3.

# Funções do meu sistema:

Cadastrar Aluno

Listar alunos

Remover Aluno

Cadastrar Curso

Listar Cursos

Remover Curso

Matricular Aluno em Curso

Listar Cursos de um Aluno

Listar Alunos de um Curso

Cancelar Matrícula
